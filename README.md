## JitPack

[JitPack](https://jitpack.io)을 활용하여 해당 레파지토리를 다른 레파지토리에서 dependency로 사용할 수 있다.

### gradle

```groovy
repositories {
    //...
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.chayoung-choi:sb-core-alpha:0.0.1'
}
```
